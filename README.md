### 目录结构

- java
  - com.springboot.demo
    - common 关键返回模型（分页，回传参数）
    - controller 控制器
    - exception 异常处理器
    - mapper dao层
      - xml
    - model 模型
    - service service 层
      - impl
    - utils 工具类
    - DemoApplication.java 启动类
  - resources 资源类
    - static 静态文件
      - css 
      - img
      - js
      - plug 前端框架
      - small 子页面
      - login.html 旧的登录页面
      - login-page.html 新的登录界面
      - main.html 主界面框架
  - application.yml 配置文件
